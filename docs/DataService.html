DataService

Data processing and spreadsheet management service for NAHS Caseload Counts application.

Provides comprehensive data operations including spreadsheet connection management, data retrieval, filtering, transformation, and caching. Handles integration with Google Sheets API and implements performance optimizations through intelligent caching.
